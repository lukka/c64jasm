# c64jasm

See [C64jasm manual](https://nurpax.github.io/c64jasm)

C64 6502 assembler in TypeScript.

## Contents

This repository contains:

- **[c64jasm DevTools](vscode/README.md)** — the c64jasm DevTools extension for VS Code, adding syntax highlighting, compile diagnostics, rich code navigation, and integrated C64 debugging.

- **c64jasm** — the C64 6502 assembler, which is also embedded into the VS Code [c64jasm DevTools](vscode/README.md) extension.
