# c64jasm

This repository contains:

- **[c64jasm DevTools](vscode/README.md)** — the c64jasm DevTools extension for VS Code, adding syntax highlighting, compile diagnostics, rich code navigation, and integrated C64 debugging.

- **[c64jasm](https://nurpax.github.io/c64jasm)** — a fork of the C64 6502 assembler, which is also embedded into the VS Code [c64jasm DevTools](vscode/README.md) extension.
